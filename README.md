## 配置

1. 将`lib`,`include`添加到对应的项目属性中，使用相对路径。
2. 因为使用了`sprintf`，在*禁用特定警告*（项目配置->配置属性->C/C++->高级）中添加4996。
3. VS运行环境必须是`x86`，因为使用的一些库是32位版本。

## 可执行文件

双击`CGFinalProject\CGFinal\Debug\setup.exe`安装后运行。
